1. Перевод модели в формат .stl
2. Установка параметров печати в OrcaSlicer. Важно! Необходимо сделать утолщенные стенки, в 2-3 слоя, а также расположить деталь так, чтобы слои были параллельны ящику, таким образом уменьшив в дальнейшем нагрузку на ручку.
